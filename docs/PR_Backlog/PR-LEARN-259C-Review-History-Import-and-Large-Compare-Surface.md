# PR-LEARN-259C - Review/History Import and Large Compare Surface

Status: Completed 2026-03-21

## Summary

Reduce the biggest current learning friction by making prior runs easy to bring
into staged curation and easy to inspect at useful size.

## Goals

- import prior outputs from history/discovered-review into curation
- give users larger side-by-side or rapid-compare inspection
- remove the need for extra image generation just to create learning evidence

## Scope

Extend:

- discovered-review surfaces
- history-import bridges
- Learning/Review large-preview surfaces

Implement:

- history-aware import into staged curation
- large compare/review viewer
- faster multi-image navigation and advancement

## Success Criteria

- users can curate old runs without regenerating images
- image review is no longer constrained by tiny previews
